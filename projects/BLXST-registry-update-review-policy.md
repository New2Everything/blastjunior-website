# BLXST Registry Update Proposal Review Policy v0

Purpose: review registry update proposals before any apply rehearsal.

This is not an auto-approval tool.

## Core Law

The review gate may produce a review report only.

It must not write registry files, website source, D1, R2, KV, Workers, Cloudflare, git, or deployment state.

## Non-Hardcoded Principle

Review dimensions and result families are non-exhaustive.

Unknown proposal shapes safe-stop into triage.
