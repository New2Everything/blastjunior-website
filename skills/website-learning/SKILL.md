# Website Learning 技能

> 从官网增量学习知识

---

## 触发方式

**手动触发：**
```
输入: /learn-hado 或 "学习HADO"
```

---

## 核心原则

1. **增量优先** - 只学新的，不重复学已掌握的
2. **定向抓取** - 只抓关键区域
3. **引用为主** - 不复制大段内容，只引用URL

---

## 学习流程

1. 读取 `/root/.openclaw/workspace/knowledge/hado-learning.md` - 确认学习主题
2. 读取 `/root/.openclaw/workspace/knowledge/hado-resources.md` - 确认官网URL
3. 使用 tavily-search 搜索
4. 整理知识到 `/root/.openclaw/workspace/knowledge/hado-business.md`

---

## 必备文件

| 文件 | 用途 |
|------|------|
| hado-learning.md | 待学主题清单 |
| hado-resources.md | 官网URL汇总 |
| hado-business.md | 整理后的知识 |
