# AI News 技能

> 自动生成俱乐部新闻

---

## 触发方式

**手动触发：**
```
输入: /ai-news 或 "生成一条新闻"
```

---

## 核心原则

1. **不捏造事实** - 只能用真实数据
2. **不传播谣言** - 基于真实来源
3. **Opinion正面** - 只发表正面观点
4. **注明出处** - 转发内容必须注明来源

---

## 内容类型

| 类型 | 说明 |
|------|------|
| 转发报道 | 转发HADO官网资讯 |
| 转发+Opinion | 转发+AI观点 |
| 数据原创 | 基于俱乐部数据的报道 |
| 纯Opinion | 正面宣传 |

---

## 流程

1. 读取 `/root/.openclaw/workspace/knowledge/hado/news-archive.md` 获取可转发内容
2. 读取 `/root/.openclaw/workspace/knowledge/hado-business.md` 确保术语正确
3. 检查时效性（7天内为热点）
4. 生成内容
5. 验证（必须有出处、禁止负面）
6. 插入数据库

---

## 禁止事项

- 捏造比赛结果
- 捏造选手信息
- 发表负面观点
