# 性能基准 SLO 参考

来源：BLXST设计白皮书

## 1. 页面体验SLO

| 场景 | 指标 | 目标SLO |
|------|------|----------|
| 首页首屏可用 | 首屏内容可交互 | ≤2.5秒 |
| 列表页滚动 | 滚动与加载 | 不明显卡顿 |
| 筛选/搜索 | 交互响应 | 有明确加载状态 |
| 详情页打开 | 主要信息出现 | ≤2.0秒 |
| 画廊列表 | 首屏缩略图出现 | ≤2.5秒 |
| 相册详情 | 首屏照片墙出现 | ≤3.0秒 |

## 2. API SLO

| API类别 | 指标 | 目标SLO |
|--------|------|----------|
| 首页聚合 | 响应时间 P95 | ≤300ms |
| 列表分页 | 响应时间 P95 | ≤500ms |
| 详情页 | 响应时间 P95 | ≤400ms |
| 搜索 | 响应时间 P95 | ≤700ms |
| 可用性 | 成功率 | ≥99.9% |

## 3. 数据层SLO

| 场景 | 指标 | 目标SLO |
|------|------|----------|
| 常用查询 | 查询耗时 P95 | ≤200ms |
| 榜单计算 | 更新时间 | 赛后≤1分钟 |
| 一致性 | 口径权威 | D1为准 |

## 4. 媒体层SLO

| 场景 | 指标 | 目标SLO |
|------|------|----------|
| 缩略图 | 单图体积 | 轻量化 |
| 中图 | 单图体积 | 适中 |
| 原图 | 加载触发 | 仅放大/下载 |
| 失败率 | 图片加载失败 | ≤0.5% |

## 5. 缓存命中目标

- 首页聚合：≥80%
- 积分榜：≥80%
- 热点详情：≥60%

## 6. 降级策略

- KV不可用 → 直接查D1
- D1慢 → 返回"上一次快照 + 更新时间提示"
- R2图片慢/失败 → 先显示占位图与文字信息
